# Sorting Visualizer

---

### About :

This is a project that visualizes some of the Sorting Algorithms.
[Go to the visualizer](./index.html)

---

#### List of Algorithms :

The following are the sorting algorithms that are visualized.

- Bubble Sort
- Heap Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Selection Sort
